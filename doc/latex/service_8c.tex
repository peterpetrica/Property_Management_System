\mbox{\hyperlink{service_8c}{models/service.\+c}} 物业管理系统 -\/ 服务管理模块实现

本文件实现了物业管理系统中与服务相关的功能，包括：
\begin{DoxyItemize}
\item 服务类型管理：添加、修改、删除和查询服务类型
\item 服务记录管理：创建服务记录、查询服务记录
\end{DoxyItemize}

服务记录可以按照员工\+ID或房屋\+ID进行查询，支持物业管理人员 对各类服务进行全面管理和统计分析。
